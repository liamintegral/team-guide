Email Workflow

5-step process for managing all client and internal communications

5-Step Email Process

Step 1: Categorize Email

  • Client Request
  • Internal Communication
  • FYI
  • Emergency

Step 2: Determine Urgency

  • Immediate response (within 1 hour)
  • Same day response
  • Within 24 hours

Step 3: Create Zoho Projects Task

If actionable work is required, create a task with:

  • Clear description
  • Due date
  • Priority level
  • Client context

Step 4: Send Acknowledgment

Reply with acknowledgment and timeline within 2 hours

Step 5: Update CRM

Log communication in CRM with summary and next steps

Client Communication

Structured approach to client calls and follow-up

Call Workflow

Pre-call (15 minutes)

  • Review recent performance data
  • Check project status in Zoho
  • Prepare key discussion points
  • Review previous call notes

During call

  • Take detailed notes using structured format
  • Ask clarifying questions
  • Confirm action items and deadlines
  • Set next communication date

Post-call (within 1 hour)

  • Upload notes to CRM
  • Create specific tasks in Zoho Projects
  • Send summary email with next steps
  • Update project timelines

Communication Schedule

Premium Clients (> $1,500/month)

Escalation Procedures

When and how to escalate issues up the chain

When to Escalate

  • Client Emergencies: Performance issues affecting business operations
  • Technical Failures: Platform outages or campaign malfunctions
  • Scope Changes: Client requests beyond current agreement
  • Client Conflicts: Relationship issues or service complaints

Escalation Responses

Emergency Escalation (Alex Morrison)

When: After-hours emergencies, critical system failures, urgent client crises

Method: Immediate phone call + follow-up email

Format: "URGENT - [Client Name] - [Brief Issue Description]"

Client Conflict Escalation (Liam Clarkin)

When: Client disputes, relationship issues, service complaints

Method: Email with full context + request for call

Format: "Client Escalation - [Client Name] - Relationship Support Needed"

Standard Scope Change Responses

Always use these phrases when clients request additional services:
  • "We'll quote on that"
  • "We'll come back to you on that"
  • Always escalate to Liam or Alex for approval before committing to new services
